# Subway-Kiosk-Program


 ### 1. Overview
 - 한국외국어대학교 소프트웨어 공학 3조 키오스크 제작
 - 서브웨이 매장에 키오스크가 없어 직접 구현해보고 싶어 만듬
 - 프로그램 시작 후 고객을 위한 주문 인터페이스/ 매장주인을 위한 주문관리, 매출관리 인터페이스로 나뉨
 - 결제 기능은 버튼말고는 구현하지 않고, 키오스크 UI작성에 포커싱함
